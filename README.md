# 🎟️ Sistema de Gerenciamento de Eventos em Java

Projeto desenvolvido em Java com foco em lógica de programação, orientação a objetos e persistência de dados em arquivo.

---

## 🚀 Funcionalidades

* Cadastro de eventos
* Listagem de eventos ordenados por data
* Participação em eventos
* Cancelamento de participação
* Persistência de dados em arquivo local (`events.data`)

---

## 🛠️ Tecnologias utilizadas

* Java
* Programação Orientada a Objetos (POO)
* Collections (List, ArrayList)
* LocalDateTime
* Manipulação de arquivos (BufferedReader / BufferedWriter)

---

## 📂 Estrutura do projeto

src/
├── model → Classe Evento
├── service → Regras de negócio
├── util → Gerenciamento de arquivos
└── Main.java → Execução do sistema

---

## ▶️ Como executar

1. Clone o repositório
2. Abra em uma IDE (Eclipse ou IntelliJ)
3. Execute a classe `Main`

---

## 💡 Funcionalidade extra

O sistema salva automaticamente os eventos em arquivo, permitindo que os dados persistam mesmo após encerrar a aplicação.
